📷 冷蔵庫の写真から買い物リストを作る（AI）

冷蔵庫の中身を撮影してアップロードすると、AIが中身を読み取り、献立プランと照らし合わせて「買い足すものリスト」を作ります。冷蔵室・野菜室・冷凍室など複数枚（最大4枚）まとめてOKです。※「基本データ」タブのAI設定が必要です（無料のGeminiキーが使えます）。

基本データ

投与量は医師の処方どおりに入力してください。このアプリが用量を提案することはありません。

⚙️ AI設定（写真解析を使う場合）

「冷蔵庫の写真から買い物リスト」「検査結果の写真から自動入力」の2機能はAIを使います。どちらかを選んでください。

📌 無料キーの取り方: aistudio.google.com にGoogleアカウントでログイン →「Get API key（APIキーを取得）」→ 作成されたキーをコピーして下に貼り付け。 クレジットカード登録は不要で、無料枠（1日250回まで）の範囲なら料金はかかりません。
📌 platform.claude.com で取得したAPIキーを貼り付けてください。解析1回あたり数円程度の利用料がかかります。

キーはこの端末のブラウザ内にのみ保存され、選んだAIの公式API以外には送信されません（バックアップのエクスポートにも含まれません）。 家族や友達とアプリを共有する場合も、キーは各自で取得してください。

📷 検査結果の写真から自動入力（AI）

血液検査の結果用紙を撮影してアップロードすると、AIが数値を読み取って下のフォームに自動入力します。読み取り結果は必ず原本と見比べてから保存してください。※「基本データ」タブのAI設定が必要です（無料のGeminiキーが使えます）。

血液検査データを登録

わかる項目だけ入力すればOKです。最新の検査結果がアドバイスに反映されます。

写真から読み取った数値も、保存前にこのフォームで自由に手直しできます。保存済みの記録は下の一覧の「編集」から修正できます。

データの保存・引っ越し

入力内容はその場で自動保存されます（画面右上に保存時刻を表示）。データはこの端末のブラウザ内にのみ保存されます（外部送信なし）。バックアップや別端末への移行にはエクスポートを使ってください。

📖 無料AIキーの取り方（5ステップ・約3分）

クレジットカードの登録は不要です。Googleアカウント（GmailのアカウントでOK）だけ用意してください。

ステップ1️⃣ ブラウザで「aistudio.google.com」を開く

スマホでもパソコンでもOK。検索で「AI Studio」と探しても見つかります。

ステップ2️⃣ Googleアカウントでログイン

普段Gmailなどで使っているGoogleアカウントでログインします。持っていない場合は「アカウントを作成」から無料で作れます。

ステップ3️⃣ 「Get API key（APIキーを取得）」を押す

ステップ4️⃣ 「APIキーを作成」を押して、できたキーをコピー

⚠️ このキーは他人に教えないでください（あなた専用の鍵です）。

ステップ5️⃣ このアプリに戻って、貼り付けて保存

「✅ 設定済み」と表示されたら完了！冷蔵庫写真・検査結果写真のAI機能が使えるようになります。
